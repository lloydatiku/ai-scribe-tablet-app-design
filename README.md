# AI Scribe Tablet App Design

Interactive tablet app prototype for an AI scribe clinical documentation workflow.

| File | Purpose |
| --- | --- |

| `uploads/workbook_extracted.txt` | Extracted workbook text used in the prototype package |

## Description

This artefact demonstrates a tablet-based AI scribe workflow for clinical documentation. It presents a governed documentation cockpit for listening, transcription, note drafting, explanation, verification, and write-back.

The prototype uses fictional/synthetic clinical data for design and research discussion.
